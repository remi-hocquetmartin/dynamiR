# Installation Guide

## System Requirements

### macOS
- **OS**: macOS 10.13 or later
- **Processor**: Intel or Apple Silicon (M1/M2+)
- **RAM**: 2GB minimum (4GB recommended)
- **Disk Space**: 500MB

---

## Installation

### macOS Application

1. **Download** the latest macOS application from the [home page](../index.md)
2. **Open** Finder and navigate to Downloads
3. **Drag** the app to Applications folder
4. **Launch** from Applications or Spotlight

✅ One-click installation - ready to use!

---

## Troubleshooting Installation

### "Application won't launch" (macOS)
- **Solution**: Right-click app → Open → Open (bypass security warning)
- First launch may take a few seconds to initialize

---

## First Launch

When you launch the application for the first time:

1. **Main window** will open with two tabs: "Matching" and "Dynamite"
2. **Help tooltips** appear on hover (marked with `?`)
3. **Sample data** files are available in the `data/` directory

👉 **Next**: Read [Getting Started](getting_started.md) to begin your first analysis!

---

## Updating

Simply download and install the new version - it will replace the old one.

---

## System Paths

After installation, files are located at:

**macOS**:
```
/Applications/DynamiR.app
```

Database and configuration files are stored in the application directory.
